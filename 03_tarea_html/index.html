Diccionario Maestro de HTML

"Introduccion al lenguaje

HTML es el lenguaje estandar utilizado para la estructurar y organizar el contenido de las paginas web mediante etiquetas.
No es programación lógica, sino una arquitectura que define elementos como títulos, párrafos, enlaces e imágenes.
Se le considera el "esqueleto" de un sitio, proporcionando la base necesaria para que el navegador interprete la información.
Para funcionar plenamente, se apoya en CSS para el diseño visual y en JavaScript para añadir interactividad y funciones complejas.
Sin HTML, la web carecería de orden y los navegadores no sabrían cómo mostrar los datos al usuario.

Etiquetas De Formato

1. La etiqueta (br)
(Break)

Sirve para crear un salto de línea simple. Función: Obliga al texto a continuar en la línea de abajo sin dejar un espacio extra entre ellas (como un "Enter" en un procesador de textos).
Uso común: Para escribir direcciones, poemas o cualquier caso donde quieras separar frases pero que sigan perteneciendo al mismo bloque o párrafo.
Dato técnico: Es una etiqueta "vacía" (no necesita cierre)
2. La etiqueta (hr) (Horizontal Rule) Sirve para crear una separación temática o un cambio de contexto.
Función: Tradicionalmente dibuja una línea horizontal que cruza la página de lado a lado.
Uso común: Para separar secciones de un artículo, indicar que termina un tema y empieza otro, o simplemente como un recurso visual de división.
Dato técnico: Al igual que (br), no requiere etiqueta de cierre.
